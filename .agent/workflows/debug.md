## /debug — Depuração

1. Identificar o módulo com problema
2. Verificar logs do servidor: `npm run dev` (terminal)
3. Verificar erros no console do navegador
4. Verificar logs do Supabase: painel > Logs > API
5. Isolar o problema (client vs server vs banco de dados)
6. Aplicar correção e verificar
